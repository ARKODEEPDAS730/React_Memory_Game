# Getting Started with Create React App
Visuospatial Memory & Interference Game
A specialized cognitive training application built with React.js, designed to challenge visuospatial working memory through interference tasks. In this game, players must memorize a generated sequence of locations on a 5x5 grid. To simulate cognitive load and test retention, a distraction task (shape matching) is inserted between every memory cue.
